## 💻 Matrix Terminal Theme

A Matrix-style green-on-black theme for macOS Terminal.

### Install

1. Download [`Matrix.terminal`](matrix-terminal-theme/Matrix.terminal)
2. Double-click the file — macOS Terminal will import it automatically
3. Open **Terminal → Settings → Profiles**, select **Matrix**, click **Default**
4. Open a new terminal window to apply

### Manual import (if double-click doesn't work)

1. Open Terminal → Settings → Profiles
2. Click the `···` menu → **Import...**
3. Select `Matrix.terminal`
4. Set as default
